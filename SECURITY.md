# Security Policy

## Supported Versions

Only the latest stable release on the `main` branch receives security fixes.

| Version | Supported |
|---|---|
| 1.1.x   | Yes |
| < 1.1   | No  |

## Reporting a Vulnerability

Linux Backup Manager manages Restic repository passwords, password files, and backup
repository access. Please **do not open a public GitHub issue** for security vulnerabilities,
since that would disclose the issue before a fix is available.

Instead, report vulnerabilities privately by email to **marcel.saager@gmx.de**.

Please include:

* A description of the vulnerability and its potential impact
* Steps to reproduce (if applicable)
* The affected version or commit

You should expect an initial response within 7 days. Once a fix is available, it will be
released as a patch version and credited in the `CHANGELOG.md` unless you prefer to remain
anonymous.
